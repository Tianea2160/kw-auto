## Functions

- 자동 로그인
- 개인 포트폴리오 페이지 검색 기능
- 수업 시간표 folding 기능
- 수업 시간표에 과제 현황 표시

## How To Use

1. 확장 프로그램을 설치한다.
    1. chrome web store에서 확장 프로그램을 설치한다.
    2. 프로젝트 파일을 다운로드해서 확장 프로그램을 설치한다.
2. 설치한 확장 프로그램의 popup에 학번과 비밀번호를 입력합니다.
3. 편하게 홈페이지를 사용하시면 됩니다.
